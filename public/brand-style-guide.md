# Étude Rapide - Brand Style Guide

## Brand Identity

### Name
**Étude Rapide** (Quick Study)

### Tagline
- **French**: "Apprenez Rapidement, Réussissez Brillamment"
- **Haitian Creole**: "Aprann Rapid, Reyisi Byen"
- **English**: "Learn Fast, Succeed Brilliantly"

### Mission
Democratize quality education across the Francophone world, with a focus on Haiti and French-speaking communities.

## Typography

### Primary Font
**Inter** (Google Fonts)
- Clean, modern, highly readable
- Excellent for multilingual content
- Supports French accents and special characters

### Font Weights
- Regular (400): Body text
- Medium (500): Subheadings
- Semibold (600): Buttons, labels
- Bold (700): Headings
- Extrabold (800): Hero text

### Font Sizes
```
Hero: 3rem (48px)
H1: 2.25rem (36px)
H2: 1.875rem (30px)
H3: 1.5rem (24px)
Body: 1rem (16px)
Small: 0.875rem (14px)
```

---

## Logo Usage

### Primary Logo
- Full color logo on white background
- Minimum size: 120px width
- Clear space: 20px on all sides

### Logo Variations
1. **Full Color**: Primary use
2. **White**: On dark backgrounds
3. **Monochrome**: Single color applications
4. **Icon Only**: Small spaces (favicon, app icon)

### Don'ts
- Don't stretch or distort
- Don't change colors
- Don't add effects (shadows, gradients)
- Don't rotate

## Iconography

### Style
- Outline style (Heroicons)
- 24px default size
- 2px stroke width
- Rounded line caps

### Usage
- Navigation: Use consistent icons
- Actions: Clear, recognizable symbols
- Status: Color-coded indicators

## Photography & Imagery

### Style Guidelines
- Bright, optimistic
- Diverse representation
- Focus on learning and success
- Authentic, not stock-looking

### Image Ratios
- Hero images: 16:9
- Course thumbnails: 4:3
- Profile photos: 1:1
- Blog headers: 21:9

---

## Voice & Tone

### Brand Voice
- **Encouraging**: Motivate learners
- **Clear**: Simple, direct language
- **Friendly**: Approachable, warm
- **Professional**: Trustworthy, credible

### Language Guidelines
- Use active voice
- Keep sentences short
- Avoid jargon
- Be inclusive

### Multilingual Considerations
- Maintain consistent tone across languages
- Respect cultural nuances
- Use local idioms appropriately

---

## Accessibility

### Color Contrast
- Minimum 4.5:1 for normal text
- Minimum 3:1 for large text
- Test with WCAG AA standards

### Typography
- Minimum 16px for body text
- Line height: 1.5-1.6
- Paragraph width: 60-70 characters

### Interactive Elements
- Minimum 44x44px touch targets
- Clear focus states
- Keyboard navigation support
